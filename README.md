# Vacation Management

Sistema de gestión de vacaciones corporativo que permite el registro, aprobación y seguimiento de solicitudes de vacaciones. Incluye gestión de empleados, periodos vacacionales y regímenes laborales, integrando roles de empleado, gerente y RRHH para un flujo de aprobación eficiente.

## Variables de Entorno

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `NEXT_PUBLIC_DOMAIN`
- `NEXT_PUBLIC_LOGIN`
- `NEXT_PUBLIC_AUTH_URL`

## Iniciar el proyecto

```bash
pnpm install
cp .env.example .env.local # Configurar variables
pnpm dev
```
